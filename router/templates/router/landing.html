Let’s talk about the project

This project was inspired by my constant need to have long URL's look attractive, short and customized. I mostly share links to social media posts with my friends and they are always long and ugly. So when it was time for me to build my Portfolio Project for Holberton School, I decided to build the cutomizable URL Shortener to satisfy my need and that of others facing the same predicament.
